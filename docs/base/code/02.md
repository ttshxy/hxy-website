# Ajax/Fetch/axios三者有什么区别
三者都是用于网络请求，但是是不同维度

Ajax是一种技术统称（异步的Javascript and XML）

Fetch是一个具体的API

axios是一个第三方库

Ajax可以通过一些API进行实现（比如通过XMLHttpRequest或者Fetch）

Fetch与XMLHttpRequest是一个级别，更方便使用且支持Promise
