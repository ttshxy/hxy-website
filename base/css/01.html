# 什么是BFC？有什么作用？

BFC 全称是 Block Formatting Context，翻译过来就是块级格式化上下文，每一个BFC都是一块独立的区域，互不影响。

# 如何触发BFC？

  • overflow不是visiable的块元素（hidden，auto，scroll）
  • display为table或者flex等
  • Position为absolute或者fixed
  • 浮动元素float不为none

# BFC的作用

# 1、避免外边距重叠（margin塌陷）

margin塌陷有两种，一种是垂直并列坍塌，一种是嵌套坍塌

# 垂直坍塌如下：

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      * {
        margin: 0;
        padding: 0;
      }
      .box1 {
        width: 200px;
        height: 200px;
        background: yellowgreen;
        margin-bottom: 50px;
      }
      .box2 {
        width: 200px;
        height: 200px;
        background: gray;
        margin-top: 40px;
      }
    </style>
  </head>
  <body>
    <div class="box1">
    </div>
    <div class="box2">
    </div>
  </body>
</html>

此时两者的边距为50px并不是90px

解决margin坍塌可以把box1与box2放置在BFC的容器中，那么box1与box2就不会互相干扰。

核心代码如下：

<style>
   .container {
        overflow: hidden;
    }
</style>
<body>
    <div class="container">
      <div class="box1"></div>
    </div>
    <div class="container">
      <div class="box2"></div>
    </div>
</body>

# 嵌套坍塌如下：

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    *{
      margin:0;
      padding:0;
    }
    .father {
      width: 100px;
      height: 100px;
      background-color: red;
    }
    .child {
      width: 30px;
      height: 30px;
      background-color: green;
      margin-top: 10px;
    }
  </style>
</head>
<body>
  <div class="father">
    <div class="child"></div>
  </div>
</body>
</html>

嵌套方向也会有margin重叠，子child的margin-top会带着father往下移动。

解决方式很简单，把father设置为BFC，子元素的布局不会对外界有影响：

.father {
      width: 100px;
      height: 100px;
      background-color: red;
      overflow: hidden;
 }

# 2、清除浮动

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      * {
        padding: 0;
        margin: 0;
      }
      .father {
        border: 1px solid red;
      }
      .child {
        width: 100px;
        height: 100px;
        background-color: green;
        float: left;
      }
    </style>
  </head>
  <body>
    <div class="father">
      <div class="child"></div>
    </div>
  </body>
</html>

当子元素设置了浮动，父布局会撑不起来，只有一条2px的线

解决方式是让父布局生成BFC

.father {
        border: 1px solid red;
        overflow: hidden;
 }

# 3、阻止元素被float元素覆盖

正常情况下float元素不占据文档流位置，会覆盖当前位置的其他元素（注意如果其他元素在前，float则不会覆盖）

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {
      padding:0;
      margin: 0;
    }
    .box1{
      width: 100px;
      height: 100px;
      background-color: red;
      float: left;
    }
    .box2{
      width: 200px;
      height: 200px;
      background-color: green;
    }
  </style>
</head>
<body>
  <div class="box1"></div>
  <div class="box2"></div>
</body>
</html>
